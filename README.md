## Comandos Usados

Clonar repositorio
```
$	git clone git@github.com:davidangarita1/repo-DevsC4.git
```

Ingresar a la carpeta
```
$	cd repo-DevsC4
```

Crear Ramas
```
$	git checkout -b Dev
$	git checkout -b Test
$	git checkout -b Issues
```

Moverse a la rama Dev
```
$ git checkout Dev
```
